# Design personality — Modelic refresh

## References I trust
- **hugeinc.com** — work-led, generous canvases, type that owns the page, no funnel theatrics.
- **impeccable.style** — taste-first, opinionated typography, comfortable being quiet.
- **wk.com** (Wieden+Kennedy) — editorial confidence, photography as the hero, no agency bombast.
- **cyber64.com** — playful, technically literate, has a sense of humor about the medium.

What these share: confidence without hubris, work over claims, type and photography doing the heavy lifting, surfaces that feel composed by a person who cares.

## What I won't tolerate
- **Stuffiness.** Anything that feels like an enterprise pitch deck. No "best-in-class", no certifications band, no LinkedIn-grade hero photography.
- **Datedness.** Skeuomorphic gradients, drop shadows on cards, generic stock photos under teal washes (the current site's hero — explicitly a thing to leave behind), Inter 300 at 47px, sharp 0px-radius CTAs that read as 2018 Squarespace.
- **Hubris.** "We are world-class." "Industry-leading." Big numbers no one asked for. Photographic close-ups of the founder. Self-congratulatory case-study language that buries the client.

If a section could appear on any agency's site verbatim, it shouldn't appear on this one.

## One rule I'm breaking — animations should be subtle
This refresh is licensed to be **expressive in motion**. Get creative:
- Page-load and scroll moments are allowed to be loud — curtain reveals, type that draws itself, the wave mark animating into existence are all fair game.
- Hover states can have personality (small but specific gestures — a CTA that pulls, type that shifts weight, an image that breathes). Generic fade-ins are the boring default to avoid.
- The Modelic wave is a *mark in motion*, not a static glyph. Treat it as kinetic by default — it should oscillate, scrub, modulate. If a static rendering is needed (favicon, print), that's the variant; the canonical mark moves.
- Section transitions can be theatrical when the content earns it (hero, case-study openers, the closing "Let's create something special together" panel). Don't sprinkle motion uniformly — pick the moments and commit.

Subtlety isn't the goal. **Specificity is.** Every animation should feel chosen — like it knows what it's saying.

## Implications for downstream skills
- Briefings: copy should be plain, partner-first, and specific. Strip any line that could be on a competitor's site verbatim.
- Wireframes: don't be afraid of asymmetric, off-grid, edge-bleeding layouts. The current site is centered and templated; the refresh shouldn't be.
- Prototype: the visual register can run cooler and more editorial than the current site. Reduce the teal photo wash to selected moments rather than every hero. Promote the wave mark into a kinetic element. Reconsider Inter 300 in display sizes — go heavier and tighter, or pair with a properly editorial display face.
- Voice gate: kill any sentence with "world-class", "best-in-class", "industry-leading", "unlock", "explosive", "transformative", "synergy", "leverage", or any single-word-superlative band.
